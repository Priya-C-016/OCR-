# OCR Web Application

## Overview
This project is an Optical Character Recognition (OCR) web application that processes uploaded images containing both Hindi and English text and returns the extracted text. It also allows users to search for keywords within the extracted text.

## Features
- Upload images for OCR processing.
- Extract text from images (supports Hindi and English).
- Search for keywords within the extracted text.
- Simple web interface using Streamlit.
